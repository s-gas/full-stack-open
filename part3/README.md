# phonebook backend

The backend is deployed on Render at <https://phonebook-h49z.onrender.com>

## API Endpoints

### Read all entries

```
GET /api/persons
```

### Read a single entry

```
GET /api/persons/{id}
```

### Create an entry

```
POST /api/persons
```

### Delete an entry

```
DELETE /api/persons/{id}
```

### Read info

```
GET /info
```
